# Release / Deploy (GitHub Pages)

This project deploys as a static site to GitHub Pages.

## Commands

```bash
npm run build:gh
npm run deploy:gh
```

## Checklist

- [ ] `npm test`
- [ ] `npm run build`
- [ ] Verify `/simple` loads and scrolls correctly
- [ ] Verify Resume opens (PDF viewer iframe mode)
- [ ] Verify Photos opens + zoom/pan works
- [ ] Verify Start menu search works and apps launch
- [ ] `npm run build:gh`
- [ ] `npm run deploy:gh`
